# Controls
***Requires an xbox controller***

- X - shoot
- A - accelerate
- B - decelerate/reverse
- Y - pause the game
- D-pad left - turn left
- D-pad right - turn right


This was made for my high school Computer Science class so please don't expect a AAA game
